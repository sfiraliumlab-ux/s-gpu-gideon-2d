$$\text{Position}(t) = \sin(t \cdot \Phi + \text{phase}_{\text{genome}})$$

## 🛠 Установка и запуск

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/your-username/s-gpu-gideon.git

```


2. **Установите зависимости:**
```bash
pip install -r requirements.txt

```


*Файл `requirements.txt` должен содержать: `streamlit`, `numpy`, `pandas`, `matplotlib`.*
3. **Запустите терминал GIDEON:**
```bash
streamlit run app.py

```
